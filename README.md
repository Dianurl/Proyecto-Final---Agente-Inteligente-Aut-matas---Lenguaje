# Proyecto-Final---Agente-Inteligente-Aut-matas---Lenguaje

Manual de usuario – Proyecto Final

Al iniciar el programa, el agente inteligente se iniciará y saludará al usuario. Posteriormente le preguntara al usuario si desea ingresar algún lenguaje o si desea salir del programa. El usuario podrá escribir lo que quiera, pero deberá de agregar la palabra “Lenguaje” o “lenguaje” para analizar; y “Salir” o “salir” para terminar el programa.
 <img width="921" height="207" alt="image" src="https://github.com/user-attachments/assets/03271407-b772-4bf1-9109-02241f77abd5" />


En el caso de pedir un lenguaje, el agente inteligente le pedirá al usuario que ingrese el lenguaje regex que quiere analizar. Posteriormente a esto el agente generara automáticamente 5 ejemplos correctos y 5 ejemplos incorrectos del lenguaje ingresado.
 <img width="921" height="395" alt="image" src="https://github.com/user-attachments/assets/dbd20209-ed59-4375-939b-4ce883050113" />


Posteriormente, el agente le preguntara al usuario si ahora le gustaría que le mostrara la tabla de transición del autómata. En este caso (y en los siguientes) el usuario podrá escribir cualquier texto, conteniendo la palabra “si” o “no” (tanto mayúscula como minúscula), para que el agente siga con el análisis. 

Caso en decir que “si”:
Siempre que se diga que sí, aparte de generar los datos en consola, también se abrirá automáticamente un .png de lo solicitado.
 
 <img width="921" height="622" alt="image" src="https://github.com/user-attachments/assets/9855ead1-6a06-4aab-ae09-04175337cc9c" />


Caso en decir que “no”:
 <img width="921" height="121" alt="image" src="https://github.com/user-attachments/assets/1f89c9d3-0711-4456-9a8b-aca8604cef8f" />
 

Al seguir con el análisis, el agente inteligente le preguntara al usuario si también le gustaría ver el grafo del autómata. En este caso, el usuario también puede decir si hacerlo o no.

Caso en decir que “si”:
Siempre que se diga que sí, aparte de generar los datos en consola, también se abrirá automáticamente un .png de lo solicitado.
 
 <img width="921" height="486" alt="image" src="https://github.com/user-attachments/assets/e57fcc56-a613-48a2-8828-9fae7cfe2e2b" />


Caso en decir que “no”:
 <img width="921" height="238" alt="image" src="https://github.com/user-attachments/assets/816f6cfb-3bc5-4183-9fb2-beb16b2d2498" />


Posterior a eso, el agente inteligente le volverá a hacer otra pregunta al usuario. En este caso le preguntará si quiere ver un resumen de todo lo obtenido por medio de un “PDF” que será simulado por una pagina HTML con todos los datos. Nuevamente, el usuario tendrá la libertad de escribir si lo quiere o no.

Caso en decir que “si”:
Siempre que se diga que sí, aparte de generar los datos en consola, también se abrirá automáticamente un .png de lo solicitado.

Por último, luego de mostrar la información, el usuario podrá escribir cualquier comentario simulando un agradecimiento. Luego el agente inteligente repetirá el bucle preguntando si el usuario quiere ingresar otro lenguaje o salir del programa.
 <img width="921" height="121" alt="image" src="https://github.com/user-attachments/assets/5f2064bf-c19b-4c7c-94c0-8bc3fc1c44aa" />


Aparte de esto, si el usuario elige ingresar otro lenguaje y escribe el mismo, el agente habrá guardado previamente los lenguajes ingresados y le dirá al usuario si quiere proseguir a pesar de ser un lenguaje duplicado. Si elige que, si se repetirá todo el proceso y si dice que no, volverá a la pregunta previa.
 <img width="921" height="171" alt="image" src="https://github.com/user-attachments/assets/71d2bcae-e99a-49a4-bfda-79dbd01b1489" />


Y cuando el usuario elija salir, se terminará el programa.
